# Portofolio

Website portofolio pribadi yang menampilkan profil, pengalaman, proyek, dan halaman kontak. Desain website dibuat dengan tema neobrutalism.
Access here : https://firzakrn.github.io/Porto/

## Bahasa yang digunakan

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome
- Vanilla JavaScript

## Struktur file

- index.html: halaman intro / landing page
- porto.html: halaman profil dan portofolio pribadi (unduh cv saya disini)
- project.html: halaman showcase proyek
- contact.html: halaman kontak dan form
- head.html: navbar/header
- style.css: styling utama dan animasi custom
- images/: aset gambar dan ikon

## Cara menjalankan

Karena project ini adalah static website, Anda bisa langsung membuka file index.html di browser.

Jika ingin dijalankan lewat local server, bisa pakai:

```bash
python -m http.server 8000
```

Lalu buka http://localhost:8000/.
